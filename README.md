
# Time_Gen

---

I've had an idea for a while that I wanted to create a tool that would build a timeline of events that I could use for documenting security investigations. The idea is that, as I find some useful piece of info, whether it be a log entry, feedback from a user, or anything else noteworthy, I would add it to the timeline. Once I'm done adding entries, the tool would output a nicely formatted document with all the entries in order of their occurence. 

My python skills are very ... rough at the moment, but this can also be a useful opportunity to learn. The idea is to build it as much as possible without AI. I'm sure I'll have AI offer syntax suggestions or explain errors as I build, but I'd like to learn as much as possible with this, and to acheive that, I really want to try to do this myself.
